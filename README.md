## JMAURA Jewelry Website

JMAURA is an online jewelry store that allows customers to browse, view, and order jewelry easily.
The platform focuses on simplicity, visual presentation of products, and fast ordering through **WhatsApp**.

## Features

1. **Product Catalog**

   Customers can browse different jewelry products available on the website.

   **Functionalities**
   - Filter products by **price**
   - Sort products by **category**
   - View **product image**
   - View **product name**
   - View **product price**
   - **Add to cart** button

2. **Product Page**

   When a user clicks on a product, they can see detailed information.

   **Information displayed**
   - Multiple **product photos**
   - **Description**
   - **Price**
   - **Add to cart** button
   - **Order via WhatsApp** button

## Essential Functionalities

### Shopping Cart

Customers can manage their selected items.

**Features**
- Add **multiple jewelry items**
- View **total price**
- Remove items from cart

### Order System

Orders are sent directly through **WhatsApp**.

## Advanced Features

### Direct Payment

Customers can pay directly using:
- **Mobile Money**
- **Bank Card**

### Customer Account (Optional at the beginning)

Customers can:
- Create an account
- View their previous orders

### Instagram Gallery

The website displays **JMAURA Instagram photos** to showcase jewelry collections.

Customers can enter their email to receive:
- New collections
- Promotions
- Updates

Customers can **save jewelry items** they like.

### Customer Reviews

Customers can leave **reviews and feedback** about products.

### Recommended Jewelry

The system suggests related jewelry.

Example:
If a customer views a **bracelet**, the website may suggest:
- Matching **rings**
- Matching **earrings**

### Promotions

The website can display discounted prices.

## Technologies (Example)

This project can be built using:
- HTML
- CSS
- JavaScript
- React or Flutter (optional)
- Firebase / Database
- WhatsApp API integration

## Project Goal

The goal of this project is to create a **modern digital store for JMAURA jewelry**, allowing customers to easily browse products and order them quickly via WhatsApp.
